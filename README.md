# Thermometer_ESP32
 Simple Thermometer using two ESP32 boards communicating with each other.

 Materials: 
 - 2x ESP32-WROOM-32D
 - 1x BME280
 - 1x 1.3' OLED IIC
 - Wires
 - BreadBoards
